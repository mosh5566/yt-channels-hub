# 📺 מרכז הערוצים שלי

לוח בקרה אחד לכל ערוצי היוטיוב שלך. כרטיס לכל ערוץ, וכפתור שפותח את **YouTube Studio של אותו ערוץ בלחיצה אחת** — גם כשהערוצים יושבים על חשבונות גוגל נפרדים.

מותקן כאפליקציה בטלפון (PWA), בלי App Store, בלי תשלום, בלי שרת. הנתונים נשמרים אצלך במכשיר בלבד.

---

## ✨ מה זה עושה

- **כרטיס לכל ערוץ** — שם, אווטאר, צבע, והמייל שמנהל אותו.
- **🎬 פתח סטודיו** — נפתח ישירות ב-`studio.youtube.com` של הערוץ הנכון, **עם החשבון הנכון** (הקסם שפותר את בעיית הריבוי חשבונות).
- **▶ פתח ערוץ** — קישור מהיר לערוץ הציבורי ביוטיוב.
- **חיפוש** מהיר כשיש הרבה ערוצים.
- **גרירה** לסידור מחדש של הכרטיסים.
- **גיבוי וייבוא** לקובץ — כי הנתונים נשמרים מקומית.

---

## 🚀 הרצה מהירה במחשב (לבדיקה)

PWA חייב להיות מוגש דרך שרת (לא לפתוח את הקובץ ישירות). הכי פשוט:

לחיצה כפולה על **`start.bat`**, או בטרמינל:

```powershell
cd "C:\Users\mosh5\Desktop\yt-channels-hub"
python -m http.server 8080
```

ואז פתח בדפדפן: **http://localhost:8080**

## 📱 התקנה בטלפון (כאפליקציה)

כדי שזה יעבוד בטלפון צריך להעלות את התיקייה לאינטרנט (HTTPS). שתי דרכים חינמיות וקלות:

### דרך א' — Netlify Drop (הכי מהיר, 30 שניות)
1. נכנסים ל-**https://app.netlify.com/drop**
2. **גוררים את כל התיקייה** `yt-channels-hub` לתוך החלון.
3. מקבלים כתובת קבועה (למשל `my-channels.netlify.app`).
4. פותחים את הכתובת בטלפון.

### דרך ב' — GitHub Pages
1. מעלים את התיקייה ל-repository ב-GitHub.
2. Settings → Pages → Branch: `main` → Save.
3. מקבלים כתובת `username.github.io/repo`.

### ואז — "הוסף למסך הבית"
- **אנדרואיד (Chrome):** תפריט ⋮ ← *Add to Home screen / התקן אפליקציה*.
- **אייפון (Safari):** כפתור שיתוף ⬆️ ← *Add to Home Screen*.

מקבלים אייקון על המסך שנפתח כמו אפליקציה אמיתית. ✅

## ➕ איך מוסיפים ערוץ

לוחצים **ערוץ חדש** וממלאים:

| שדה | חובה | הסבר |
|------|------|------|
| שם הערוץ | ✅ | איך הוא יוצג בכרטיס |
| מייל גוגל | ✅ | החשבון שמנהל את הערוץ — מה שמבטיח פתיחה בחשבון הנכון |
| מזהה/קישור | מומלץ | מדביקים כתובת ערוץ, `UC…` או `@שם`. בלי זה ייפתח הסטודיו של החשבון בלי לכוון לערוץ ספציפי |
| אימוג'י + צבע | לא | לזיהוי מהיר ויזואלית |

### איך מוצאים את מזהה הערוץ (Channel ID)?
`studio.youtube.com` ← **Settings** ← **Channel** ← **Advanced settings** → מעתיקים את ה-`Channel ID` שמתחיל ב-`UC…`.

> 💡 הכי מומלץ למלא מזהה — כך הכפתור מקפיץ ישר לסטודיו של הערוץ המדויק, גם אם לחשבון יש כמה ערוצים.

## 📊 נתונים חיים (אופציונלי)

הדשבורד יכול להציג **מנויים, צפיות, מספר סרטונים ותמונת ערוץ** אמיתיים בכל כרטיס. צריך לזה מפתח חינמי של YouTube (חד-פעמי, ~2 דקות):

1. נכנסים ל-[console.cloud.google.com](https://console.cloud.google.com) ויוצרים פרויקט.
2. מפעילים את **YouTube Data API v3** (חיפוש למעלה → Enable).
3. **APIs & Services → Credentials → Create Credentials → API key** — מעתיקים.
4. באפליקציה: תפריט ⋯ → מדביקים את המפתח בשדה **"YouTube API Key"** → "שמירת מפתח".

הנתונים נמשכים אוטומטית ומתרעננים כל 30 דקות (או בלחיצה על "רענן נתונים"). המכסה החינמית (10,000 ליום) מספיקה בקלות. המפתח נשמר במכשיר שלך בלבד.

> 💡 כדי שזה יעבוד, לכל ערוץ צריך למלא מזהה/קישור (`UC…` או `@שם`) — לא רק מייל.

## 🔒 פרטיות

הכול רץ בדפדפן שלך. אין שרת, אין מסד נתונים, אין מעקב. הערוצים נשמרים ב-`localStorage` של הדפדפן — לכן כדאי לעשות **ייצוא גיבוי** מדי פעם (תפריט ⋯).
