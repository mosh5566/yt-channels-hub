הגדרות מתקדמות איך יוצרים Client ID משלי?
  1. console.cloud.google.com ← הפעל Google Drive API.
  2. OAuth consent screen ← External ← הוסף את עצמך כ-Test user.
  3. Credentials ← OAuth client ID ← Web application.
  4. Authorized JavaScript origins: https://mosh5566.github.io
השגת YouTube API Key (חינם, ~2 דקות):
  1. נכנסים ל-console.cloud.google.com ויוצרים פרויקט חדש (או בוחרים קיים).
  2. מחפשים למעלה YouTube Data API v3 ולוחצים Enable.
  3. בתפריט: APIs & Services ← Credentials ← Create Credentials ← API key.
  4. מעתיקים את המפתח ומדביקים למעלה, ולוחצים "שמירת מפתח".

🔒 מומלץ: לוחצים על המפתח ← Restrict key ← מגבילים ל-YouTube Data API v3, ותחת Website restrictions מוסיפים mosh5566.github.io/*.
